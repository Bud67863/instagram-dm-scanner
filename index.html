Get your data from Instagram

Open Instagram’s download page ↗

Log in if asked. If that link doesn’t open the right page: instagram.com → Settings → Accounts Center → Your information and permissions → Download your information.

  1. Choose Download or transfer information
  2. Select your Instagram account (if asked)
  3. Pick Some of your information → check Messages only → Next (messages-only exports are ready much faster)
  4. Choose Download to device
  5. Set Date range — Last year is plenty for this scan — and set Format to JSON ← required, the HTML format won’t work here
  6. Hit Create files, then watch for Instagram’s email (usually minutes for messages-only)
  7. Download the .zip from the email and drop it on this page — no need to unzip

You press Send on each message — nothing is ever sent for you. With auto-fill set up (below), Open DM lands in the thread with your message already typed. Without it, the message is on your clipboard — paste and send.

⚡ Set up auto-fill — skip the paste (one-time, ~1 minute)
  1. Install the Tampermonkey browser extension
  2. Then click here to install the auto-fill script — Tampermonkey shows an install screen, hit Install
  3. That’s it. Open DM ↗ now opens the thread with your message already typed in the box — you just press Send. The script only types; it can’t send.
